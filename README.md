# 독립언론 솔루션 — ALT프로덕션 안내 페이지

ALT프로덕션이 공급하는 독립언론·마을 경제 플랫폼 솔루션(스탠다드 / 프로페셔널)의 안내 페이지입니다.

본 저장소는 [index.html](./index.html) 한 장으로 구성된 정적 페이지이며, Vercel에 그대로 배포해 운영합니다.

## 주요 안내 내용

- 스탠다드 / 프로페셔널 두 등급 비교
- 일시불 가격 (얼리버드 / 파운더 / 정상가)
- 월정액 2년 약정 옵션
- 어보전 시스템·음성 기사 작성·기사 인쇄 최적화 등 Pro 단독 기능
- 도입 절차 및 문의 채널

## 관련 저장소

- [local-news-standard](https://github.com/dreamq2018-lab/local-news-standard) — 스탠다드 등급 코드
- [local-news-pro](https://github.com/dreamq2018-lab/local-news-pro) — 프로페셔널 등급 코드
